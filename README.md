# Simple XML+XSL Viewer
A browser-based tool to view XML files styled with XSLT.

- No server required (Pure Client-side).
- Drag & Drop support.
- Print-ready CSS (hides UI when printing).

XMLファイルを XSLT スタイルシートで変換し、ブラウザ上でプレビュー・印刷するためのシンプルなクライアントサイド・ツールです。

## 概要
このツールは、ブラウザ上でファイルを読み込ませることで、サーバーを立てることなくかんたんに変換結果を確認できるように設計されています。

## 主な機能
- サーバー不要: HTMLファイル1つで動作します。外部ライブラリに依存しないため、オフラインでも使用可能です。
- ドラッグ＆ドロップ対応: XMLファイルとXSLファイルを同時にドロップするだけで、自動的にスタイルを適用します。
- 印刷最適化: 印刷時にはビューアのUI（ドロップエリアやボタン）を自動で非表示にし、変換後の文書のみをきれいに印刷できます。

## 使い方
1. simple-xml-xsl-viewer.html をブラウザで開きます。
1. 表示したい XMLファイル と XSLファイル を同時に選択し、画面上のエリアにドラッグ＆ドロップします。
   または、「ファイルを選択」ボタンから両方のファイルをまとめて選択してください。
1. 変換結果が表示されたら、右上の「印刷」ボタンからPDF保存や紙への印刷が可能です。

すべてのXML+XSLが問題なく表示されることを保証するものではありません。
